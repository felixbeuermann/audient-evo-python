# AI Agent & Automation Policy

This repository is maintained by a solo developer and is intended for human-driven development and collaboration. To maintain the integrity of the codebase and ensure high-quality contributions, the following policies apply to all automated agents, Large Language Models (LLMs), and AI-assisted workflows.

---

## 1. Automated Scraping & Training

* **No Unauthorized Training:** Data in this repository may not be used to train, fine-tune, or validate AI models without explicit written consent from me.
* **Scraping Limits:** Automated scraping for the purpose of data mining or commercial redistribution is strictly prohibited.

## 2. Autonomous Contributions

* **No Bot-Generated Pull Requests:** I do not accept Pull Requests (PRs) opened entirely by autonomous AI agents. Any PR must be initiated, reviewed, and submitted by a human contributor who takes full responsibility for the code.
* **Issue Spam:** Automated tools may not be used to mass-open issues or comments. Such activity will be flagged as spam and result in an immediate block.

## 3. AI-Assisted Development (for Humans)

I recognize that tools like Copilot or ChatGPT are part of the modern workflow. However, if you choose to use them, the following rules apply:

* **Human Oversight Required:** If you use AI to help write code for a contribution, you must manually review every line. *"The AI wrote it"* is not a valid excuse for bugs, security vulnerabilities, or poor architecture.
* **Attestation:** I reserve the right to ask contributors to disclose if a significant portion of a PR was generated by AI.
* **Licensing Awareness:** Ensure that any AI-generated code does not violate third-party licenses or inject proprietary code from other sources into this project.

## 4. Enforcement

Failure to comply with these guidelines will result in:

* Immediate closure of the PR or Issue.
* Reporting of the account to GitHub for bot-like behavior/spam.
* A permanent block from this repository.

---

> **Note to Developers:** As a solo maintainer, my time is limited. I highly value genuine human creativity, problem-solving, and critical thinking. Let's keep this codebase crafted by humans, for humans. Thank you for respecting my time!
